# 仿映客直播
 
# 前言

本项目是本人练习项目，仿映客直播，用到的知识点有：  

- Bottom Bar+Fragmentation+DataBinding构建开发框架 
- 七牛云实现推流(PLDroidMediaStreaming)和拉流(PLDroidPlayer) 
- 环信聊天室实现直播间聊天
- BaseRecycleViewAdapterHelper实现多种布局
- SmartReFreshLayout实现下拉刷新和上拉加载
- ViewAnimator实现聊天弹幕和礼物特效
